# ScoreBoard Dashboard

A full-stack leaderboard management application built using React, Vite, Express.js, MongoDB Atlas, and Tailwind CSS.

## Features

* Create, Read, Update, and Delete Players
* Global Ranking System
* Search Players
* Sort by Highest and Lowest Score
* Pagination
* Dynamic Statistics Dashboard
* CSV Export
* Toast Notifications
* Delete Confirmation Modal
* Dark / Light Theme
* Responsive Design
* REST API Integration
* Custom React Hooks
* Context API Theme Management

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* Axios
* React Hot Toast

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## Installation

### Clone Repository

```bash
git clone <repository-url>
cd scoreboard-dashboard
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

Start Backend:

```bash
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```

Backend runs on:

```text
http://localhost:5000
```
